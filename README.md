💬 MERN Chat App

A beginner-friendly real-time chat application built with the MERN Stack and Socket.io. This project demonstrates how modern chat applications work, including authentication, real-time messaging, online user tracking, and state management.

🚀 Features
🔐 Authentication & Authorization with JWT
💬 Real-time messaging using Socket.io
🟢 Online user status tracking
👤 User signup and login system
🎨 Modern and responsive UI with Tailwind CSS & DaisyUI
⚡ Global state management with Zustand
🐞 Robust error handling on both client and server
📱 Responsive design for desktop and mobile devices
🚀 Free deployment setup

🛠️ Tech Stack
Frontend
* React
* React Router DOM
* Tailwind CSS
* DaisyUI
* Zustand
* Axios
* Socket.io Client
  
Backend
* Node.js
* Express.js
* MongoDB
* Mongoose
* Socket.io
* JWT Authentication
* bcrypt.js
